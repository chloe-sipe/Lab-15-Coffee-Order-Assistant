# Lab-15: Coffee Order Assistant
This program is used to order from a coffee shop menu

## How to Run
1. Open the program in your IDE
2. Look over the menu and indicate your order by typing the item number
3. If you are a cafe member, answer "Y." If not, answer "N"
4. View your reciept

## Example Output
--- MENU ---
1. Espresso - $3.50
2. Latte - $4.50
3. Cappuccino - $5.00

What would you like to order? (1-3) 2

Are you a member? (Y/N) N

--- RECIEPT ---

Drink: Latte

Price: $4.50
